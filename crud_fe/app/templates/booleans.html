Working with Booleans in Python

        
            INPUT:
            x = 5
            y = 8
            
            print("x == y:", x == y)
            print("x != y:", x != y)
            print("x < y:", x < y)
            print("x > y:", x > y)
            print("x <= y:", x <= y)
            print("x >= y:", x >= y)

            Output:
            x == y: False
            x != y: True
            x < y: True
            x > y: False
            x <= y: True
            x >= y: False
